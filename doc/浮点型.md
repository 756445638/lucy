### **类型(float)**

float类型由关键字'float'标识。
默认值: 0
范围: 从 1.401298464324817E-45f 到  3.4028234663852886E38f
字面值可以由以下的方式初始化

~~~
    a := 0.0
    a = 0.1f
    a = 0.1F
~~~

<br/><br/>


### **类型(double)**

double类型由关键字'double'标识。
默认值: 0
范围: 从 4.9E-324到 1.7976931348623157E308
字面值可以由以下的方式初始化

~~~
    a := 0.0D
    a = 0.1d
~~~
