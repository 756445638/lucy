
### **类型(byte)**

byte类型由关键字'byte'标识。
默认值: 0
范围: 从 -2^7 到 2^7-1(-128 到 127)
字面值可以由以下的方式初始化

~~~
	a := 1b
	a = 'a'
    a = 1B 
~~~
